5月6日下午，我们将在菩提谷二期的山上竹林中，举行一场竹林宴，做野火饭，品野山茶，当然还有美景和好酒。欢迎来聚！

\n\n

 

\n\n

以前，每到立夏，小孩子最高兴，可以烧野火饭，这是自古以来就形成的风俗。带上家人朋友或约上几个玩伴，到野外空地，挖一个小坑搭几块砖当灶台，在锅子里放上糯米，去溪边淘干净。几个人分头行动相互配合，有人挖笋，有人采蚕豆、豌豆，有人捡柴。把笋和咸肉切成丁，和剥好的豆子一股脑儿倒进锅里。再放些蒜苗，切些腌制的咸菜放进去，滴上菜油，加进盐，水要没过所有的材料。满满一锅货色，看着就诱人，散着山野的气息。

\n\n

 

\n\n

五月的乡间，蚕豆、豌豆越长越鼓，竹园里的笋一天天蹿高，地里的大蒜也茎粗叶绿。天气干燥，枯枝落叶遍地都是，野火饭所需的材料完全能就地取材了。老宋说，在山上做一次野火饭吧，别浪费了这么好的天气和环境。于是，开始呼朋唤友，约聚山林，有人说要有茶，有人说要有酒，七嘴八舌，一顿野火饭，最后谋划成了一场竹林宴。也学一回古人，集于竹林之下，肆意酣畅，畅叙幽情。
